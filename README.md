

# Tecnologías utilizadas

- React
- TypeScript
- Vite
- CSS puro

---

# Clonar el repositorio

```bash
git clone https://github.com/asGerath/react-layout-hitse.git
```

---

# Entrar al proyecto

```bash
cd react-layout-hitse
```

---

# Instalar dependencias

```bash
npm install
```

---

# Levantar el proyecto

```bash
npm run dev
```

Después de ejecutar el comando, Vite mostrará una URL local similar a esta:

```bash
http://localhost:5173
```

Abrir esa URL en el navegador.

---

# Scripts disponibles

## Ejecutar entorno de desarrollo

```bash
npm run dev
```

## Generar build de producción

```bash
npm run build
```

## Previsualizar build

```bash
npm run preview
```
